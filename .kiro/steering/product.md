# YIWU EXPRESS - Product Overview

## What We Build

YIWU EXPRESS is a global trade and logistics platform connecting buyers with suppliers and manufacturers in Yiwu, China. The platform enables cross-border ecommerce, product sourcing, and simplified international shipping.

## Core Offerings

- **B2B Marketplace**: Connect directly with Yiwu manufacturers and distributors
- **Product Catalog**: Browse millions of products across diverse categories
- **Multi-Currency Pricing**: Support for international currencies and real-time exchange rates
- **Purchase Management**: Order tracking, variants, and bulk purchasing
- **User System**: Buyer profiles, wishlist, preferences, and order history
- **Mobile App**: Native React Native experience via Expo for iOS/Android
- **Web Platform**: Full-featured Next.js web application

## Target Users

- International buyers sourcing from China
- SME entrepreneurs purchasing in bulk
- Drop-shippers and resellers
- B2B procurement teams

## Key Platform Features

- Multi-currency and language support
- Product attribute system (colors, sizes, specifications)
- Dynamic category management
- Hero slider with animated promotions
- Wishlist functionality
- Purchase order system with variants
- User authentication and authorization
- Admin dashboard for product and category management
- Image upload and CDN delivery
- Real-time cart synchronization

## Technical Scope

- **Monorepo**: Web (Next.js), Mobile (React Native), Backend (Node.js)
- **Database**: PostgreSQL with Prisma ORM
- **Frontend**: React 18 with TypeScript, Tailwind CSS
- **Auth**: JWT-based authentication
- **Payment**: Stripe and PayPal integration
- **Notifications**: Push notifications via Expo
