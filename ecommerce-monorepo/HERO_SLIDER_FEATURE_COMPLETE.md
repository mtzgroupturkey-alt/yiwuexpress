## 💡 COMMON WORKFLOWS

### **Workflow 1: Create New Slide**
```
1. Click "Add Slide"
2. Fill Content tab (title, description)
3. Fill Media tab (upload images)
4. Fill Settings tab (duration, motion type)
5. Click "Create Slide"
6. Toggle active when ready
```

### **Workflow 2: Duplicate & Customize**
```
1. Find slide to duplicate
2. Click Copy button (📋)
3. Confirm duplication
4. Click Edit on new slide
5. Update title (remove " (Copy)")
6. Change specific content
7. Activate when ready
```

### **Workflow 3: Reorder Slides**
```
1. Drag slide by grip handle (⋮⋮)
2. Drop in new position
3. Click "Save Order"
4. Order updated!
```

### **Workflow 4: Seasonal Campaign**
```
1. Create base slide
2. Duplicate for each season
3. Edit duplicates:
   - Change images
   - Update text
   - Adjust colors
4. Activate current season only
5. Deactivate others
```

## 🎓 PRO TIPS

### **1. Use Duplicates for Efficiency**
Don't create from scratch! Duplicate similar slides and edit.

### **2. Mix Motion Types**
- 80% Slide/Fade (reliable)
- 20% Zoom/Flip/Rotate (special)

### **3. Keep Inactive Until Ready**
New slides default to inactive - perfect for preparing campaigns.

### **4. Test Different Versions**
Duplicate → Change CTA → Test which performs better.

### **5. Organize by Season**
Create seasonal slides, keep inactive, activate when needed.
