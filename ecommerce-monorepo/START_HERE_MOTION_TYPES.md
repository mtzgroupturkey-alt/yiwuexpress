# 🚀 START HERE - MOTION TYPES

**Add unique animations to each hero slider slide in 5 minutes!**

---

## ⚡ WHAT IS THIS?

Your hero slider now supports **6 different animation styles**! Each slide can have its own unique motion type.

**Available Animations:**
- 🎬 **Slide** - Classic horizontal sliding
- ✨ **Fade** - Elegant crossfade
- 🔍 **Zoom** - Dynamic zoom in/out
- 🔄 **Flip** - 3D flip effect
- 🌀 **Rotate** - 360° rotation  
- 📍 **Scale** - Pop from center

---

## 🎯 QUICK SETUP (3 Commands)

**Step 1:** Stop your dev server (Ctrl+C)

**Step 2:** Run these commands:

```bash
cd ecommerce-monorepo/web
npx prisma generate
npx prisma db push
npm run dev
```

**Step 3:** Done! ✅

---

## 🎨 HOW TO USE

1. Open: `http://localhost:3001/admin/settings/hero-slider`
2. Click **Edit** on any slide
3. Go to **Settings** tab
4. Find **Motion Type** dropdown
5. Select animation (e.g., "Zoom In/Out")
6. Click **Update Slide**
7. Visit homepage to see it!

---

## 🎬 TRY THIS NOW

**Quick Demo:**

1. Edit **Slide 1** → Set Motion Type: **Zoom**
2. Edit **Slide 2** → Set Motion Type: **Fade**
3. Go to homepage
4. Watch the different animations! 🎉

---

## 💡 WHICH MOTION TO USE?

**General Use:** Slide or Fade (80% of slides)  
**Special Announcements:** Zoom or Scale  
**Creative Brands:** Flip or Rotate  
**Elegant Content:** Fade only  
**Exciting Content:** Zoom
