## 🎬 Custom Animations

### **CSS Animations Added** (`globals.css`)

#### 1. **slide-in**
```css
Duration: 0.3s
Easing: ease-out
Usage: Success toast notification
Effect: Slides from right (100% → 0) + fade
```

#### 2. **fade-in**
```css
Duration: 0.4s
Easing: ease-out
Usage: Page sections, expandable content
Effect: Fades in + moves up (10px)
```

#### 3. **pulse-glow**
```css
Duration: 2s
Loop: infinite
Usage: Attention-grabbing elements
Effect: Shadow expands and fades
```

#### 4. **bounce-subtle**
```css
Duration: 1s
Loop: infinite
Usage: Emphasis elements
Effect: Gentle 5px bounce
```

### **Animation Classes**
- `.animate-slide-in` - Toast notifications
- `.animate-fade-in` - Page load, reviews
- `.animate-pulse-glow` - CTAs (optional)
- `.animate-bounce-subtle` - Badges (optional)

