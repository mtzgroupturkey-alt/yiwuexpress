## 🚀 **Quick Start Instructions**

### **1. Setup Database & Admin User**
```bash
cd web
npm install
npx prisma generate
npx prisma db push
npx prisma db seed
```

### **2. Start Development Server**
```bash
npm run dev
```

### **3. Access Admin Dashboard**
- **URL:** `http://localhost:3000/auth/login`
- **Admin Email:** `admin@yiwuexpress.com`
- **Admin Password:** `admin123`
