## 🚀 How to Use Categories

### 1. Access Categories
```
URL: http://localhost:3001/admin/categories
Location: Admin sidebar → Categories
```

### 2. Create Main Category
1. Click **"Add Category"** button
2. Enter name (e.g., "Electronics")
3. Slug auto-generates
4. Leave parent as "None"
5. Check "Active"
6. Click **"Create"**

### 3. Create Subcategory
1. Click **"Add Category"** button
2. Enter name (e.g., "Laptops")
3. Select parent (e.g., "Electronics")
4. Check "Active"
5. Click **"Create"**

### 4. Assign to Products
1. Go to **Products** → **Edit** product
2. Select category from dropdown
3. Save product
