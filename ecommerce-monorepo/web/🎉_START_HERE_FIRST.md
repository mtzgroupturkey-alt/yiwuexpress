### Manage Products

2. See all 5 sample products
3. Click **Add Product** to create new
4. Click **Edit** icon to modify existing
5. Delete products (protected if has orders)

### Process Orders
1. Click **Orders** in sidebar
2. See all orders (if any)
3. Click an order to see details
4. Update status via dropdown
5. Create shipments with tracking

1. Click **Wholesale** in sidebar
2. View wholesale inquiries
3. Create quotes with pricing
4. Convert accepted quotes to orders

### Configure Countries
1. Click **Countries** in sidebar
2. See USA and China (sample data)
3. Click **Add Country** to create new
4. Edit countries to set tax & customs
5. View shipping rates per country
