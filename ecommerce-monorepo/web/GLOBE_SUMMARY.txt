================================================================================
  CUSTOMIZATION OPTIONS
================================================================================

1. Change Globe Size
   Edit: components/footer.tsx
   Find: className="w-80 h-80"
   Change to: className="w-96 h-96" (larger)

2. Adjust Animation Speed
   Edit: components/footer.tsx
   Find: speed={0.002}
   Change to: speed={0.005} (faster)

3. Change Opacity
   Edit: components/footer.tsx
   Find: opacity-30
   Change to: opacity-50 (more visible)

4. Add More Locations
   Edit: components/footer.tsx
   Add to globalNetworkMarkers array:
   { id: "paris", location: [48.85, 2.35], name: "Paris", users: 800 }

5. Change Colors
   Edit: components/ui/cobe-globe-interactive.tsx
   Find: baseColor, markerColor, glowColor
   Adjust RGB values (0-1 range)

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Globe not visible
Solutions:
  - Check you're on desktop (width > 1024px)
  - Hard refresh: Ctrl + Shift + R
  - Check browser console for errors
  - Verify cobe installed: npm list cobe

Problem: Globe not rotating
Solutions:
  - Check browser supports WebGL
  - Try different browser (Chrome recommended)
  - Check JavaScript is enabled
  - Look for console errors

Problem: Performance issues
Solutions:
  - Globe is already hidden on mobile
  - Uses GPU acceleration automatically
  - Try closing other browser tabs
  - Update graphics drivers
